# Playwright JS

# Installation
install git, nodejs and vscode (optional) from company portal
in git bash:
`git clone https://www.github.com/xzovyAtWork/playwrightjs.git`

then: 

`npm init -y`
`npm install`
`npx playwright install chromium`
